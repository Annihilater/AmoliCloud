<h1 align="center">AmoliCloud</h1>

> 😊一款轻量级的私有云存储系统，支持本地、阿里云OSS、腾讯云COS存储，提供简单的文件列表、上传下载、管理等功能。

### 注意事项/Attention
-----
#### 环境
**运行环境**：PHP 5.6+ 推荐7.x，cURL()支持

**开发环境**：PHPstudy、PHP 7.2、Nginx、Windows

#### 兼容性
支持IE9及以上的现代浏览器，并且已在 Chrome、firefox、IE11 等浏览器测试使用正常
